# Task-01-VardanDagar
Repository for task 01


SmartBot is a simple rule-based chatbot built using Python. It responds to predefined user inputs using conditional statements (if-elif-else) and maintains an interactive conversation through a continuous loop.

This project demonstrates the fundamentals of chatbot development, control flow, decision-making logic, and user interaction in Python.

**Features**
● Handles greetings such as Hi, Hello, and Hey
● Answers common questions about AI, Python, and internships
● Responds to thank-you messages
● Displays timestamps with every chatbot response
● Runs continuously until the user exits
● Supports exit commands like exit, bye, and quit
● Beginner-friendly implementation

**Technologies Used**
● Python 3
● Conditional Statements(if-elif-else)
● Loops
● Functions
